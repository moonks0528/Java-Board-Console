# Java Console Board

Java 콘솔 환경에서 동작하는 간단한 게시판 프로그램입니다.

Java 기초 문법과 객체지향 구조를 이해하기 위해 만든 프로젝트입니다.

---

## 프로젝트 소개

콘솔에서 실행되는 게시판 프로그램으로  
게시글을 작성하고 조회, 수정, 삭제할 수 있습니다.

ArrayList와 객체지향 구조를 활용하여  
게시판 기능을 구현했습니다.

---

## 주요 기능
2026/3/22
- 게시글 작성
- 게시글 목록 조회
- 게시글 상세 조회
- 게시글 수정
