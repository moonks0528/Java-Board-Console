# Java Console Board
컴퓨터 SW 전공 학습을 위해 제작한 Java 콘솔 게시판 프로젝트입니다.

Java 콘솔 환경에서 동작하는 간단한 게시판 프로그램입니다.

Java 기초 문법과 객체지향 구조를 이해하기 위해 만든 프로젝트입니다.

---

## 프로젝트 소개

콘솔에서 실행되는 게시판 프로그램으로  
게시글을 작성하고 조회, 수정, 삭제할 수 있습니다.

Java의 클래스 구조와 ArrayList를 활용하여  
게시판 기능을 구현했습니다.

---

## 주요 기능

- 게시글 작성
- 게시글 목록 조회
- 게시글 상세 조회
- 게시글 수정
- 게시글 삭제
- 프로그램 종료

---

## 실행 화면
===== 게시판 =====

1. 글 작성
2. 글 목록
3. 글 조회
4. 글 수정
5. 글 삭제
6. 종료
